Questions and Answers:

  1. What is XML?
    A text-based syntax designed to describe, deliver, and exchange structured information.
  2. How is XML used?
    It is used to extend the power of HTML by seperating data from presentation. It allows developers to create whatever tags they need to describe their information.
  3. What does XML look like? (Provide an example.)
    XML looks like using tags like book or author in a bookseller's website's code to easily identify what information is being applied to relevant elements.
  4. What is JavaScript?
    JavaScript is an object-based, client side scripting language interpreted by a web browser.
  5. How is JavaScript related to HTML and CSS?
    JavaScript is related to HTML and CSS because it is a client side scripting language which causes it to be embedded within the HTML file that is then modified by the CSS file and ultimtely interpreted by the browser.
  6. What are some common JavaScript elements, properties, methods, and functions?
    (Provide at least 1 example of each)
    Textboxes are an element,the background color of the document is a property, the command to display an alert message is a method, and invoking a displayAlerts() function would be a function is JavaScript.
